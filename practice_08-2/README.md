## Задание 2. Менеджер задач To-Do List на SQLite
Описание:

Консольный менеджер задач с сохранением в tasks.db
Таблица tasks:
- id INTEGER PRIMARY KEY AUTOINCREMENT
- text TEXT NOT NULL
- deadline DATE
- priority TEXT CHECK(priority IN ('высокий','средний','низкий')) DEFAULT 'средний'
- completed BOOLEAN DEFAULT 0
- created_at DATETIME DEFAULT CURRENT_TIMESTAMP

Функционал:
1.	Добавить задачу
2.	Показать все задачи (с сортировкой по дедлайну и приоритету)
3.	Отметить выполненной
4.	Удалить задачу
5.	Показать просроченные задачи (deadline < текущая дата и completed=0) — выделить красным (colorama)
6.	Поиск по тексту
7.	Статистика: сколько всего / выполнено / просрочено
