## Задание
Описание:

Выберите любой пост в ВКонтакте и напишите скрипт для парсинга данных людей, лайкнувших этот пост, и построения статистики по полу и возрасту на полученных данных. Формат получившейся статистики (словарь, тип dict):
```py
{
  "post_id": 31231,
  "age": {
    "0-18": 26,
    "19-35": 100,
    "36-50": 45,
    ">50": 12,
    "unknown": 129
  },
  "sex": {
    "male": 147,
    "female": 125,
    "unknown": 58
  }
}
```
