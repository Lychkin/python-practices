Организация процесса тестирования:
1. 	Планирование: Определите гипотезу: "Версия B сайта повышает конверсию по сравнению с версией A". Установите уровень значимости α=0.05, мощность теста 80%. Рассчитайте необходимый размер выборки (используйте онлайн-калькулятор или формулу из scipy).
2. 	Сбор данных: Симулируйте данные (или используйте реальные).
3. 	Анализ: Проведите t-тест, рассчитайте доверительные интервалы.
4. 	Интерпретация: Сделайте выводы и визуализируйте.
5. 	Документация: Запишите результаты в отчёт.

Код для выполнения (используйте модуль scipy.stats для статистики):
```py
import numpy as np
from scipy import stats
import pandas as pd

# Шаг 1: Симуляция данных (группа А: контрольная, группа В: тестовая)
np.random.seed(42)  # Для воспроизводимости
group_a = np.random.binomial(1, 0.1, 1000)  # Конверсия 10%
group_b = np.random.binomial(1, 0.12, 1000)  # Конверсия 12%

# Шаг 2: Рассчёт средних и интервальных оценок
mean_a = np.mean(group_a)
mean_b = np.mean(group_b)
ci_a = stats.t.interval(0.95, len(group_a) - 1, loc=mean_a, scale=stats.sem(group_a))  # Доверительный интервал для А
ci_b = stats.t.interval(0.95, len(group_b) - 1, loc=mean_b, scale=stats.sem(group_b))  # Для В

print(f"Средняя конверсия A: {mean_a:.4f}, C1: {ci_a}")
print(f"Средняя конверсия B: {mean_b:.4f}, CI: {ci_b}")

# Шаг 3: A/B-тест (t-тест для независимых выборок)
t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"T-статистика: {t_stat:.4f}, p-value: {p_value:.4f}")

# Шаг 4: Вывод
if p_value < 0.05:
    print("Разница значима: Версия В лучше.")
else:
    print("Разница не значима.")
```
