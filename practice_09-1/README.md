## Задание 1
Описание:

Зарегистрировать тестовый счётчик Яндекс.Метрики для произвольного сайта (можно использовать учебный лендинг или существующий сайт организации). Настроить права доступа для приложения (OAuth токен) и зафиксировать идентификатор счётчика и токен в конфигурационном файле .env.

## Задание 2
Описание:

Написать скрипт на Python, который с использованием HTTP запроса к API отчётов Яндекс.Метрики получит число визитов, просмотров и уникальных посетителей за последние 7 дней по выбранному счётчику. Результат вывести в консоль в виде таблицы и сохранить в CSV файл.

## Задание 3
Описание:

Модифицировать скрипт так, чтобы он позволял задавать период отчёта (дата «с» и «по») через параметры командной строки и автоматически обрабатывал возможные ошибки API (неверный токен, нет доступа к счётчику, некорректные даты), выводя понятные пользователю сообщения.
